## Install and run

Install [canvas-sketch-cli](https://github.com/mattdesl/canvas-sketch) globally first:

```
npm install canvas-sketch-cli -g
```

Run the sketch you need:

```
canvas-sketch-cli src/SKETCH_NAME.js --open
```
